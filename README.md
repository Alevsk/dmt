# Direct MinIO Tunnel

## Usage `dmt`

```
Usage of ./dmt:
  -routes string
        default access routes file (default "/etc/dmt/routes.json")
  -ca-cert string
        CA certificates (default "/etc/dmt/ca.crt")
  -tls-cert string
        TLS certificate (default "/etc/dmt/tls.crt")
  -tls-key string
        TLS key (default "/etc/dmt/tls.key")
```
