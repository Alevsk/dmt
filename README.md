# route35

## Usage `route35`

```
Usage of ./route35:
  -routes string
        default access routes file (default "/etc/route35/routes.json")
  -ca-cert string
        CA certificates (default "/etc/route35/ca.crt")
  -tls-cert string
        TLS certificate (default "/etc/route35/tls.crt")
  -tls-key string
        TLS key (default "/etc/route35/tls.key")
```
